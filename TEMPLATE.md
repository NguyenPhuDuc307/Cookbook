# [Tên công thức]

## Vấn đề
Mô tả ngắn gọn vấn đề hoặc tình huống cần giải quyết.

## Giải pháp
Giải thích cách tiếp cận và giải quyết vấn đề.

## Ví dụ

### Cách 1: [Tên phương pháp]
```[ngôn ngữ]
// Code ví dụ
```

**Giải thích:**
- Giải thích từng bước hoặc điểm quan trọng

### Cách 2: [Tên phương pháp] (tùy chọn)
```[ngôn ngữ]
// Code ví dụ khác
```

**Giải thích:**
- Giải thích phương pháp này

## Lưu ý
- Các điểm cần chú ý khi sử dụng
- Ưu và nhược điểm
- Các trường hợp sử dụng phù hợp
